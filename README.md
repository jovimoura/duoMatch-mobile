## Installation

### Clone

```
git clone https://github.com/jovimoura/duoMatch-mobile
```

### Access the folder

```
cd <name-folder>
```

### Install dependencies

```
npm i
```

### Start the app

```
npm run start || expo start
```

### Port:

```
exp://192.168.0.12:19000
```
